# Mad Scientist App

An app to control the JAWS Rover as built by the [Mad Scientist Lab](https://github.com/javaplus/MadScientist)

Currently Deployed to:
- iOS: [Mad Scientist Controls](https://apps.apple.com/us/app/mad-scientist-controls/id6739670168)
- Android: Side Load latest APK from [Releases](https://github.com/javaplus/mad_scientist_app/releases)
- Web: https://javaplus.github.io/mad_scientist_app/index.html
